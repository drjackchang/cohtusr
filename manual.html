第一章 · 系統簡介
系統簡介

USafe 是專為國立臺北護理健康大學設計的校園緊急應變系統，整合緊急求救、急救志工派遣、全校安全通報等功能，讓師生在緊急狀況下能夠快速求救、接收通知並回報安全狀況。

主要功能
🚨
一鍵 SOS 緊急求救
選擇事件類型，精準回報位置，即時通知急救員與校安中心
🛡️
急救先鋒（CFR）系統
具備急救證照的師生可登記成為認證急救員，接收附近求救通知
📢
全校安全通報
管理員發布警報，全校師生收到通知並回報安全狀況
💙
AED 位置地圖
校園內 AED 位置一目瞭然，緊急時快速導航前往
📋
離線防災指南
CPR、AED、止血、地震、火災應變指南，無網路也可查看
💚
LINE 即時通知
綁定 LINE 後，警報直接發送至 LINE，可在 LINE 內直接回報
系統網址
https://usafe-drjackchang.vercel.app
PWA 支援

1

2.2 初次設定（Onboarding）

首次登入後，系統會引導您完成初始設定，建議花 2-3 分鐘完成。

功能介紹頁面
身份選擇頁面

步驟一：滑動查看系統功能介紹

步驟二：選擇您在學校的身份（學生 / 教職員 / 其他）

LINE 綁定碼畫面
LINE 綁定成功畫面

步驟三：綁定 LINE（強烈建議）

💡 為什麼要綁定 LINE？ 綁定後，緊急警報會直接發送到您的 LINE，即使沒有開啟 App 也能即時收到通知，並可在 LINE 內直接按鈕回報安全狀況。
1
點擊「取得綁定碼」，系統產生 6 位數綁定碼（10 分鐘內有效）

收到「✅ LINE 綁定成功！」即完成

第四章 · 急救志工功能
急救志工功能

具備急救證照的師生，經管理員審核通過後升級為「認證急救員」，可接收附近求救通知並即時前往救援，實踐「急救先鋒（Community First Responder）」精神。

4.1 上傳急救證照

點擊首頁右上角頭像 → 個人資料 → 登記急救證照：

1
選擇證照類型（EMT-1 / EMT-2 / EMT-P / BLS / AED / CPR / ACLS / STB）
2
填寫發照機構（例如：衛生福利部、台灣紅十字會）
3
填寫證照到期日
4
上傳證照圖片或 PDF
5
按「提交審核」，等待管理員審核（通常 1-3 個工作天）
💡 審核通過後，您的身份會升級為「認證急救員」，首頁會出現志工待命開關。
4.2 接收求救通知
首頁顯示求救通知卡片

上線時，附近有人發送 SOS，首頁會出現求救通知卡片，顯示事件類型、位置、時間及已前往的急救員人數。

4.3 前往救援流程
前往救援頁面
器材選擇畫面
1
點擊「前往救援」進入救援頁面，確認事件位置
2
勾選攜帶的器材（甦醒球 BVM / AED / 急救包 / 止血帶 STB / 氧氣設備 / 擔架）
3
抵達現場後按「我已抵達現場」
抵達後選項 Modal
4
選擇現場狀況：✅ 已接觸求救者 / ❓ 未找到求救者 / 🆘 需要更多人支援
5
救援結束後按「結束救援」返回首頁
📝 救援頁面底部有快速捷徑：📋 應變指南、🗺️ AED 地圖、📞 校安中心，隨時可查閱或撥打。

第六章 · 常見問題
常見問題
Q：登入失敗怎麼辦？

請確認使用 Safari 或 Chrome 開啟系統，不要在 LINE 內直接點擊連結。LINE 內建瀏覽器不支援 Google 登入。

Q：沒有收到推播通知？

請確認已允許通知權限：

  • iOS：設定 → Safari → 通知 → 允許
  • Android：設定 → 應用程式 → Chrome → 通知 → 允許
Q：LINE 沒有收到通知？
  • 確認已完成 LINE 綁定（設定頁顯示「已綁定」）
  • 確認已加 USafe 校園緊急通報（@102ienck）為好友
  • 確認 LINE 通知未被靜音
Q：如何重新綁定 LINE？

設定頁 → LINE 綁定狀態 → 點擊重新綁定 → 取得新綁定碼 → 傳給 Bot 完成綁定。

Q：找不到「後台」按鈕？

後台按鈕只有管理員帳號才會顯示。如需管理員權限，請聯繫系統管理員。

Q：上傳證照後多久審核？

由管理員人工審核，通常在 1-3 個工作天內完成。審核通過後系統會更新您的身份為「認證急救員」。

Q：SOS 誤觸怎麼辦？

在確認畫面點「取消求救」即可取消。為避免誤觸，SOS 按鈕需要滑動才能觸發。

Q：手機加入主畫面後和一般 App 有什麼不同？

加入主畫面後（PWA），USafe 會以全螢幕方式開啟，體驗接近原生 App，同時支援防災指南離線閱讀。
